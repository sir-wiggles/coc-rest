# coc-rest

A coc-post or vim-rest-console plugin but with more features.

## Install

`:CocInstall coc-rest`

## Keymaps

`nmap <silent> <C-l> <Plug>(coc-coc-rest-keymap)`

## Lists

`:CocList demo_list`
